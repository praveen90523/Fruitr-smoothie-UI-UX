# 🥤 Fruitr Smoothie UI/UX

A modern, premium, and responsive smoothie landing page built using **React**, **Vite**, and **Tailwind CSS**. The project focuses on delivering an elegant user experience with glassmorphism, smooth animations, interactive UI elements, and a visually appealing design inspired by modern product landing pages.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/Fruitr-Smoothie.git
```

### Navigate to Project

```bash
cd Fruitr-Smoothie
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
